Search Engine Optimization

Keywords

Spent time brainstorming terms and phrases that people may use when searching for your site. These terms are what describes you website or business.

Title Page

A descriptive page title which includes your company name or website name will help your site market itself. The page title is often displayed in the SERP and saved when a visiter bookmarks your page. Do not use the exact same title for each page of your website, include keywoeds in the page title that are appropriate for the page.

Heading Tags

Use structured tags, h1, h2, ect to organize your content. If appropriate include some keywords in the text contained within the heading tags. Some search engions will give rhigher list position if key words are used in the title page or heading. However, do not spam keywords, meaning repeating the words over and over whithin the content of the page. Search engines continue to become more sophisticated and spamming can prevent your page from being listed if it percieves you are trying to cheat the system.

description

Write a few sentences about your website or busienss that makes it feel special and why would want to visit. The description should be inviting and intresting so that someone would chose your site.

Description Meta Tag

A self-contained tag that is placed in the head section of a web page and is displayed on the SERP by some search engions, such as Google.

Linking

Verify that all hyperlinks are working, Each page on your website should be reachable by a text hyperlink and should be descriptive. Do not use phrases shuch ans more info or click here. Should use keywords as approporate.

Images and Multimedia

Search engine robots do not see the text within your images. Configure meaningful alt text for inages and influde relevant keywords.

Valid Code

While it is not required by search engine that your code be validated, well structured and valid code is more easily processed by search engine robots which could help your placement in the search engion.

Content of Values

Your website shoudl contain high-quality, well organized content that is of value to your vistitors.

Additional Techniques for SEO

Map Your Site

An HTML site map is a web page with a map of the site that contains hierarchial list of hyperlinks to the major pages on your website. This can be helpgul for website visitors and assists search engine robots as they follow hyperlinks on your site. An XML site map is an XML file that is used by search engines only, not visitors. An XML site map provides the following information about each page to a search engine:

  • date that each page was last modified
  • priority of how frequently each page changes
  • priority level for each page

You can use an outiline sitemap generator to create a sitemap file.

Link Popularity

Search ongines rate websites based on the number of sites that link to a particular website and the quality of those sites. Link popularity can determine its order in the SERP.

SMO

Make it easy for your current and potential website visitors by creating content of value that is sharable. Such as providing links to Twitter, Facebook, and Pintrest.
